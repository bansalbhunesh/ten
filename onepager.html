Loop index maps 1:1 → table row tr[i+1] (row 1 is the header) → sheet row i+1. Cells are read whole (country links are span-wrapped on some rows), then trimmed. Parse number casts the year and acts as a row-validity gate; Detour appends only when all fields are non-empty; a per-row error just skips to the next index.

Library conformance — 6 of 9 steps, nothing else

The API call — Sheets values.append via Postman

POSThttps://sheets.googleapis.com/v4/spreadsheets/{{SPREADSHEET_ID}}/values/Sheet1!A:D:append
    ?valueInputOption=RAW&insertDataOption=INSERT_ROWS
HeadersAuthorization: Bearer {{ACCESS_TOKEN}} · Content-Type: application/json AuthPostman OAuth 2.0 (Auth Code) · scope .../auth/spreadsheets 
"range": "Sheet1!A:D", "majorDimension": "ROWS",
"values": [[ {{year}}, "{{winner}}", "{{score}}", "{{runners_up}}" ]]

→ 200 · updates.updatedRange = "Sheet1!A{{index+1}}:D{{index+1}}"   RAW keeps en-dash scores from becoming dates; INSERT_ROWS only ever adds.
